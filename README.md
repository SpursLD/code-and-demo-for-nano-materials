# SRGAN-based joint super-resolution and denoising for mitigating geometric and topological biases in fine-grained electron backscatter diffraction images
1. This repository contains the Srgan code, trained parameter files, and two demos. 
2. It enables super-resolution of EBSD images with noise by running test_images.py.
3. The training dataset can be requested from the author after the article is published.
4. requirement
   torch 1.10.2, torchvision 0.11.3, PIL 8.3.1, pandas 0.25.3, tqdm 4.63.0
